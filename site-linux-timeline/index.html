History of Linux

Reece Gilbert

  • The Precursors (Unix and more)

  • 1969

  • Unix Operating System

    The Unix operating system was conceived and implmented by Ken Thompson and Dennis Ritchie of AT&T Bell Laboratories.

  • 1970

  • Unix Released

    Unix was released and later licensed out resulting in a number of both academic and commerial unix variants.

  • 1971

  • UNIX Programmer's Manual Published

    The UNIX Programmers Manual was published in the "man page" format still used to this day.

  • 1973

  • Unix was rewritten in C

    Unix was rewritten in the higher-level language C. Being written in C made Unix portable to other computers with only relatively small amounts of machine-dependent code being replaced. The availability and portability of unix would come to caused wide spread adoption of the OS.

  • 1977

  • The Berkely Software Distribution (BSD) was developed

    BSD was developed by the Computer Systems Research Group from UC Berkely. BSD contained Unix code that would later result in a lawsuit.

  • 1983

  • Richard Stallman started the GNU project

    Richard Stallman created the GNU project with the goal of creating a free UNIX-like operating System. The GNU General Public License (GPL) was created as part of his work on the GNU project.

  • 1985

  • Intel 80386 aka. i386 released

    The first x86 microprocessor with a 32-bit instruction and a memory management unit with paging was released by intel.

  • 1986

  • "The Design of the UNIX Operating System" published

    "The Design of the UNIX Operating System" was published By Maurice J. Bash of AT&T Bell Labs.

  • 1987

  • MINIX released

    Minix was a Unix-like system intended for academic use, released by Andrew S. Tanenbaum. The system was to exemplify the principles covered in his book, "Operating Sytems: Design and Implementation". MINIX's 16bit design was not well adapted to the 32-bit features if the increasinly cheap and popular Intel 386 architecture.

  • Linux (And Beyond)

  • 1991

  • Linus Torvalds begins on Linux

    Linus Torvalds began a project that would later become the linux kernal while studying computer science at the University of Helsinki.

  • Linus announces work on Linux

    On August 25 Linus announced his system in a Usenet posting to the newsgroup "comp.os.mixix."

  • Linux Gets Its Name

    Linux was originaly known as Freax (a portmanteau of "free", "freak" and "x") by Linus. During the upload to a FTP server (to facilitate development) Ari Lemmke, a volunteer administrator for the FTP server, changed the project name to Linux as he dislike the current project name "Freax".

  • 1992

  • X Window System ported to Linux

    Orest Zborowski ported the X Window System to Linux, allowing Linux to support a graphical user interface for the first time.

  • Linux released under the GPL

    In December Linus published linux version 0.99 under the GNU General Public License.

  • 1996

  • Linux gets it's Mascot

    Linus announces that there would be a mascot for linux. This mascot would be a punguin named Tux. Linux decided on a pungiun when he was bitten in a little penguin at the National Zoo & Aquarium in Canberra Australia. The name Tux is a derivative of Torvald's UniX, along with being short for tuxedo, a suit with simular color to that of a penguin.
